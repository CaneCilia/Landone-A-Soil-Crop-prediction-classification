# About

This prototype demonstrates the following objectives:

- AI-based soil classification from an image
- Rule-driven soil property estimation
- Crop recommendation based on soil type and environmental data
- A clean Gradio interface with documentation and about pages

## Motivation

Helping growers and agronomists make better decisions by combining soil observations with local conditions.

## Technologies used

- Python
- Gradio for the user interface
- OpenCV and NumPy for image processing
- Simple rule-based logic for recommendations
